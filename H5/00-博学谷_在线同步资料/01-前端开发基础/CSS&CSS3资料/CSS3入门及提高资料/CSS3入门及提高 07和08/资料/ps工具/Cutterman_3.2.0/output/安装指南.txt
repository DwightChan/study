
# 安装步骤

1. 打开PS，菜单 文件->脚本->浏览
2. 选择当前文件夹下的Install.jsx文件，接着会打开一个文件夹
3. 删除该文件夹下的cutterman目录（如果有）
4. 复制安装包的cutterman目录到该打开的文件夹
5. 重启PS


# 卸载步骤

3. 删除该文件夹下的cutterman目录
